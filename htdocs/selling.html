RESPONSE

Step 4: Send SMS Verification Code


                        
HEADER X-Coins-Api-Token: 

                        
POST http://woc.reference.genitrust.com/api/sendVerification/

                        

                    
                        
RESPONSE

Step 5: Verify Ad


                        
HEADER X-Coins-Api-Token: 

                        
POST http://woc.reference.genitrust.com/api/verifyAd/

                        

                    
                        
RESPONSE

Step 6: Update Ad Rate

%

                        
HEADER X-Coins-Api-Token: 

                        
PUT http://woc.reference.genitrust.com/api/v1/ad/

                        

                    
                        
RESPONSE

Step 7: Incoming Payments

                        
HEADER X-Coins-Api-Token: 

                        
GET http://woc.reference.genitrust.com/api/v1/incomingOrders/

                    
                        
RESPONSE

* Confirm Payment

                        
HEADER X-Coins-Api-Token: 

                        
POST http://woc.reference.genitrust.com/api/v1/incomingOrders/<orderId>/confirmDeposit/

                        

                    
                        
RESPONSE

* Deny Deposit

                        
HEADER X-Coins-Api-Token: 

                        
POST http://woc.reference.genitrust.com/api/v1/incomingOrders/<orderId>/invalidateDeposit/

                        

                    
                        
RESPONSE
