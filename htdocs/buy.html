WOC API DEMO
  • Buy Crypto
  • Sell Crypto
API Endpoint:  

Step 0: Discovery



                        
POST http://woc.reference.genitrust.com/api/v1/discoveryInputs/

                        

                    
                        
RESPONSE

Step 1: Get Offers



                            
    GET http://woc.reference.genitrust.com/api/v1/discoveryInputs/<authToken>/offers/
    
                        
    
                    
                            
    RESPONSE

    Step 2: Create Hold


                            
    POST http://woc.reference.genitrust.com/api/v1/holds/
    
                        
    
                    
                            
    RESPONSE

    Step 3: Capture Hold


                            
    HEADER X-Coins-Api-Token: 
    
                        
    POST http://woc.reference.genitrust.com/api/v1/holds/<holdId>/capture/
    
                        
    
                    
                            
    RESPONSE

    Step 4: Confirm Deposit

                            
    HEADER X-Coins-Api-Token: 
    
                        
    POST http://woc.reference.genitrust.com/api/v1/orders/<holdId>/confirmDeposit/
    
                        
    
                    
                            
    RESPONSE

    * Cancel Order

                            
    HEADER X-Coins-Api-Token: 
    
                        
    DELETE http://woc.reference.genitrust.com/api/v1/orders/<holdId>/
    
                        
    
                    
                            
    RESPONSE

    * Get Orders

                            
    HEADER X-Coins-Api-Token: 
    
                        
    GET http://woc.reference.genitrust.com/api/v1/orders/
    
                    
                            
    RESPONSE
